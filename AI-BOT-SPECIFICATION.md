## 🤖 AI Bot Configuration

### Current Setup (v2.0 — Enhanced)
- **Engine:** Groq + multi-model support (5 models with auto-failover)
- **Default Model:** Llama 3.3 70B (free tier)
- **Fallback Chain:** Llama 3.1 8B → Mixtral 8x7B → Gemma 2 9B → Llama 3.2 90B
- **Rate Limit:** 14,400 requests/day, 500K tokens/day (Groq free tier)
- **Local Knowledge Base:** Enhanced keyword-matching (100+ QA pairs, weighted scoring, negation detection, context awareness)
- **System Prompt:** Stored in `src/constants/ai-knowledge-enhanced.ts` (NEW — enhanced)
- **API Route:** `src/app/api/ai-bot/route.ts` (rewritten with 4-layer decision engine)
- **UI Component:** `src/components/shared/AiAssistant.tsx` (uses new enhanced imports)

### Decision Engine Architecture (4 Layers)
1. **Enhanced Smart Matching** — Weighted keyword scoring with context memory (covers 95%+)
2. **Groq AI Response** — Any of 5 models, auto-switching on failure
3. **Intent Classification** — Regex pattern fallback (7 intent types)
4. **Context-Aware Politeness** — Never says "I don't know" coldly
